📖 Quick Setup (click to collapse)
  1. Go to Notion Integrations → New integration.
  2. Give it a name (e.g. "LinkedIn Saver"), select your workspace, and click Submit. Copy the Internal Integration Token (starts with ntn_).
  3. In Notion, create or open a database. You can use any property names you like — configure the exact names in the Advanced: Custom Property Names section below after saving your token/DB ID.
  4. Click ⋯ → Connections → invite your new integration to the database.
  5. Copy the Database ID from the browser URL (the long string after the workspace name).
  6. Paste both values above and click Save Settings, then Test Connection.

Tip: The first time you save a profile, the extension will create a new page in the chosen database with the extracted fields.
